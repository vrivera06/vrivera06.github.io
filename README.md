# 🌐 Personal Portfolio — Victor Rivera Portillo

A personal portfolio website built with pure HTML, CSS, and JavaScript. Live at [vrivera06.github.io](https://vrivera06.github.io).

---

## 🧰 Technologies Used

- HTML
- CSS (Grid, Flexbox, Animations)
- Vanilla JavaScript
- Google Fonts — Instrument Serif · DM Mono · Geist

---

## ✨ Features

- Dark editorial design inspired by remix.run
- Animated hero section with large serif typography
- Scrolling tech stack marquee
- Sections for Experience, Projects, Skills, Education, and Contact
- Scroll reveal animations on every section
- Custom cursor with hover interactions
- Fully responsive for mobile

---

## ⚙️ Process

I wanted a portfolio that felt intentional and designed, not like a generic template. I took inspiration from remix.run — bold typography, dark backgrounds, and tight grid layouts. Everything is written from scratch with no frameworks or libraries, keeping the site fast and easy to deploy.

The main challenge was building smooth scroll animations and a custom cursor purely in vanilla JavaScript without relying on any external packages.

**What I learned:**
- How to build advanced CSS animations and transitions from scratch
- Scroll-triggered reveal effects using the Intersection Observer API
- Deploying and hosting a static site with GitHub Pages
